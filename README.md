## Usage

Clone it!

```
git clone https://github.com/hannesivask/worldwise.git
```

Create a .env.local file for the fake user

```
  VITE_FAKE_USER={
   "name":"",
   "email":"",
   "password":"",
   "avatar":"https://i.pravatar.cc/100?u=zz"
   }
```

Install npm dependencies

```
npm i
```

Run npm script:

```
npm run dev
npm run server
```
